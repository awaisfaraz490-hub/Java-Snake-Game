# Java Snake Game

A classic Snake Game developed in Java using Swing and Object-Oriented Programming.

## Features
- Start Menu
- Arrow Key Controls
- Score Counter
- Game Over Screen
- Restart Option

## Technologies Used
- Java
- Swing
- OOP Concepts

## How to Run
1. Open project in NetBeans.
2. Run SnakeGame.java.
3. Press ENTER to start.
